# Sistem-Kasir-Klinik-Persalinan---Python-Dasar
Ini adalah proyek aplikasi konsol berbasis Python yang dirancang untuk membantu administrasi klinik persalinan dalam mencatat data pasien dan menghitung biaya penginapan serta penanganan secara otomatis. Proyek ini saya buat untuk mempraktikkan logika pemrograman dasar namun krusial.
